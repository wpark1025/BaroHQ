## Legal Checklist

- Allowed licenses: MIT, Apache-2.0, BSD-2-Clause, BSD-3-Clause, ISC
- Blocked licenses: GPL, AGPL, SSPL, EUPL
- No copyrighted assets without license
- No API keys or secrets in source code
- No PII in logs or debug output
- Attribution required for open-source dependencies
- DMCA-safe content generation only
