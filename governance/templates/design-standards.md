## Design Standards

- shadcn/ui inspired components
- Tailwind CSS only, no inline styles
- Slate gray palette (slate-50 through slate-950)
- One accent color per team
- System font stack (ui-sans-serif, system-ui)
- text-sm for body, text-xs for labels
- 4px spacing grid
- rounded-lg default border radius
- Consistent hover/focus states
- Dark mode: slate-900 bg, slate-100 text
